# MACROPAD
This is my take on the HackPad Macropad from HackClub, It is my first time making a custom pcb or macropad.

This Macropad has 9 buttons, 2 mini leds, and a capacitive touch button

# Features
-3d printed case
-2 leds to show status/ layer
-customisable layers using kmk
-capacaitive touch button to swap layers.
